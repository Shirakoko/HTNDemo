# 项目说明

本项目是一个基于 **Unity 2022.3.47f1c1** 开发的应用程序。以下是项目的详细说明和使用指南。

------

## 项目概述

- **开发引擎**：Unity 2022.3.47f1c1
- **第三方库**：DOTWEEN（用于动画和过渡效果）
- 分支说明
  - **Assets 分支**：包含项目的完整资源文件（Assets 文件夹）。
  - **Scripts 分支**：仅包含项目的脚本部分。

------

## 运行方式

### 1. 下载 Windows 版本
